# AI-News-Agent


AI-News-Agent is an intelligent news automation system.

It automatically:

- Crawls news from websites
- Stores articles into SQLite database
- Uses LLM models to summarize news
- Generates AI-powered news reports
- Provides web interface and API


## Features


### News Crawling

Supported sources:

- BBC Chinese
- AIBase


Technologies:

- Requests
- BeautifulSoup
- Playwright


### AI Summary


Supports:

- DeepSeek API
- Qwen API
- OpenAI compatible API


The system automatically generates:

- News summary
- Importance score
- Keywords
- Category analysis



### Database


Using:

SQLite


Database stores:

- Title
- URL
- Source
- Content
- Summary
- Keywords



### Web Interface


Provides:

- News dashboard
- AI analysis results
- API interface
